Станция метро «Автозаводская», выход в город из последнего вагона по направлению движения поездов из центра. Из стеклянных дверей по переходу поверните налево. Выйти на улицу Мастеркова по лестнице справа, идти прямо до светофора, на пересечении c ул. Ленинская Слобода повернуть налево. Далее прямо 200 м до входа в БЦ "Омега-Плаза". Зайти внутрь здания, после турникетов налево, по лестнице на второй этаж. Вход в бизнес-центр осуществляется по предъявлению паспорта.
